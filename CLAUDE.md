# CLAUDE.md

## Planning

Any request to plan, design, or think through a task before implementing it MUST follow the `planning` skill (`.claude/skills/planning/SKILL.md`). Invoke it before doing anything else — do not improvise a plan format.

In short: research externally, then inspect the codebase, then write the plan to `docs/planning/<task-name>.md`. A planning session changes no code.
